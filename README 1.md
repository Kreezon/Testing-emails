A simple and efficient **Email Validation Web App** built using **Streamlit**, **Requests**, **BeautifulSoup**, and **ThreadPoolExecutor**.

## Features
- Validate emails from CSV files or text input
- Real-time progress bar
- Fast multithreaded validation
- Filter displayed columns
- Export results as CSV
- Clean UI with Streamlit

## Installation
```
pip install -r requirements.txt
```

## Run
```
streamlit run app.py
```
